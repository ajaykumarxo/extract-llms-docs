## 📚 How to Use extract-llms-docs

1. **Open the Application:** Launch extract-llms-docs from your applications menu or desktop shortcut.
2. **Add a Site:** Enter the URL of the website you want to extract documentation from.
3. **Configure Options:** Choose your desired settings like export format.
4. **Start Extraction:** Click the 'Extract' button and let the application do the work.
5. **Access Your Files:** The extracted documentation will be saved in your selected format in the specified directory.
